How QuickField Works

Get started in minutes and save hours of manual work

Install the Extension

Add QuickField to your Chrome browser from the Chrome Web Store with just one click.

Navigate to Salesforce

Open your Salesforce org where you want to create custom fields.

Open QuickField

Click the QuickField icon in your Chrome toolbar to open the extension.

Configure Your Fields

Enter the object name, field type, and labels (or upload an Excel file) and click create.

Monitor Progress

Watch as QuickField creates your fields and provides a detailed success/failure report.

Privacy & Security

Your data security is our top priority

Data Handling

QuickField operates entirely within your browser and only communicates with your Salesforce org and our secure backend server for field creation. Your Salesforce session ID is never stored or logged.

Server Processing

For Excel file processing, files are temporarily uploaded to our secure server where they are processed and immediately deleted after field creation is complete.

No Data Retention

We do not store any of your Salesforce data, metadata, or credentials. All processing is done in-memory and no sensitive information is persisted.

Current Limitations

Be aware of these current constraints

  • Only supports custom objects and customizable standard objects
  • Requires Chrome browser (other browsers not supported yet)
  • Excel upload only accepts .xlsx format files
  • Field-level security must be configured separately for non-admin profiles
  • Currently supports field creation but not updates to existing fields
